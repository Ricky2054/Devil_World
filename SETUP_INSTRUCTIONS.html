You're seeing "MetaMask not detected!" because the game is running directly from your file system (file:// protocol). Browser security policies prevent MetaMask from being detected when running from local files.

You need to run the game through a web server instead of opening the HTML file directly. Here are several ways to do this:

üöÄ Method 1: Use the Provided Server Scripts (Recommended)

I've created server scripts for you:

For Windows: Double-click start-server.bat
For Mac/Linux: Run ./start-server.sh in terminal

These scripts will automatically start a local server and open the game in your browser.

üêç Method 2: Python (if installed)

üì¶ Method 3: Node.js (if installed)

üîß Method 4: VS Code Live Server Extension

If you have VS Code:

  1. Install the "Live Server" extension
  2. Right-click on index.html
  3. Select "Open with Live Server"

üåê Method 5: Other Web Servers

You can use any web server software:

  • XAMPP: Place files in htdocs folder
  • WAMP: Place files in www folder
  • MAMP: Place files in htdocs folder
  • IIS: Place files in wwwroot folder

üéÆ After Starting the Server
